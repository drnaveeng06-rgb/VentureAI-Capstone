# Venture.AI

AI-powered business validation engine, startup co-founder platform for Flowzint AI Hackathon 2026, developed by Dr. Naveen Gokul R.

## Description

AI-powered startup co-founder platform that helps founders validate ideas, analyze markets, evaluate competitors, design business models and generate investor-ready strategies using Generative AI.

## Features

- Startup opportunity discovery
- Market sizing (TAM/SAM/SOM)
- Revenue model generation
- Competitor analysis
- SWOT analysis
- Go / Iterate / Stop recommendation
- Roadmap generation
- Investor pitch generator
- Founder Command Center

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Gemini API
- Chart.js

## Modes

- Demo Mode – Explore the platform using sample startup data.
- Live AI Mode – Generate personalized startup analysis using the Gemini API.

## How to Run

1. Clone the repository.
2. Open `index.html` in your browser.
3. Choose Demo Mode or Live AI Mode.
4. For Live AI Mode, enter your Gemini API key and choose among the various Gemini API models based on your search requirements.

## Venture.AI Analysis Pipeline

### Stage 1 – Opportunity Discovery
Identifies the core problem, target users, industry, market timing, and hidden opportunities.

---

### Stage 2 – Business Model & Market Validation
Generates TAM/SAM/SOM estimates, revenue architecture, competitive analysis, MVP roadmap, team requirements, and estimated startup cost.

---

### Stage 3 – Venture Evaluation
Evaluates the startup using AI scoring, SWOT analysis, Venture Radar, and Go / Iterate / Stop recommendation.

---

### Stage 4 – Execution Roadmap
Creates a phased implementation roadmap with milestones, KPIs, deliverables, timelines, and execution priorities.

---

### Stage 5 – Investor Pitch Generator
Generates a complete investor pitch, presentation slides, elevator pitch, and potential investor Q&A.

---

### Stage 6 – Founder Command Center
Provides a consolidated executive dashboard combining opportunity analysis, market insights, execution strategy, founder priorities, and final recommendations.
